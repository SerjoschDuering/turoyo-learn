# Turoyo Grammar Basics

## Sentence Structure

Turoyo follows a **verb-initial** pattern, typically **VSO** (Verb-Subject-Object) for transitive clauses, though word order is flexible due to the lack of case marking on nouns. Topicalized subjects can precede the verb.

```
Basic pattern:  VERB - SUBJECT - OBJECT
Example:        Griše  u-gawro  u-kṯowo
                read   the-man  the-book
                "The man read the book"
```

In practice, SVO order also occurs frequently, especially in spoken diaspora Turoyo and under influence from European languages.

## Gender System

Turoyo has **two grammatical genders**: masculine (m.) and feminine (f.).

- Gender applies to nouns, adjectives, verbs, articles, and pronouns
- Most nouns ending in **-o** are masculine: gawro (man), bayto (house)
- Most nouns ending in **-to/-ṯo** are feminine: aṯto (woman), ḥoṯo (sister)
- Adjectives must agree in gender: gawro rabo (big man) vs. aṯto rabṯo (big woman)

## Number

Two numbers: **singular** and **plural**.

### Plural Formation Patterns

| Pattern | Singular | Plural | Meaning |
|---------|----------|--------|---------|
| -o → -e/-one (m.) | gawro | gawrone | man/men |
| -to → -oṯe/-yoṯe (f.) | aṯto | aṯṯoṯe | woman/women |
| -o → -e (m.) | kalbo | kalbe | dog/dogs |
| Loanword + -at | | + -at | (Arabic/Turkish loans) |
| Irregular | bayto | bote | house/houses |

## Definite Articles

Turoyo uses **prefixed definite articles** that agree in gender and number.

| | Masculine | Feminine | Plural |
|---|-----------|----------|--------|
| **Definite** | u- | i- | a- / an- |
| **Indefinite** | ḥa (one) | ḥḏo (one) | (no article) |

```
u-gawro    = the man
i-aṯto     = the woman
a-gawrone   = the men
ḥa gawro   = a man
ḥḏo aṯto   = a woman
```

These derive historically from demonstrative pronouns. Compare Classical Syriac: u-malko (the king), i-malëkṯo (the queen), a-malke (the kings).

## Verb System

The Turoyo verb system is aspect-based, built on stems derived from old participles.

### Tense/Aspect Overview

| Tense | Formation | Example (verb: griše "to read") |
|-------|-----------|--------------------------------|
| **Present continuous** | ko- + verb stem | ko griš-ono (I am reading) |
| **Past (preterite)** | verb + -wa- infix | griš-wa-no (I was reading) |
| **Future** | gd- + present | gd griš-ono (I will read) |
| **Imperative** | bare stem | grëš! (read!) |

### The ko- Prefix (Present/Habitual)

The **ko-** prefix marks ongoing or habitual action:

```
ko griš-ono     = I read / I am reading
ko griš-at      = you read
ko griš-o       = he reads
ko grëš-o       = she reads
ko griš-ina     = we read
ko griš-utu     = you (pl.) read
ko grëš-i       = they read
```

### Copula (To Be)

| | Present | Past |
|---|---------|------|
| 1st sg. | -no | -wayno |
| 2nd sg. | -at | -wayt |
| 3rd sg. | -yo (m.) / -yo (f.) | -wa |
| 1st pl. | -na | -wayna |
| 2nd pl. | -utu | -waytu |
| 3rd pl. | -ne | -wayne |

```
Eno ṭowo-no       = I am good
Hat ṭowo-at        = You are good
Hiye ṭowo-yo       = He is good
Aḥna ṭowe-na       = We are good
```

### Existence and Possession

| Concept | Positive | Negative |
|---------|----------|----------|
| There is/are | kit / kito | layt / layto |
| To have | kitl- + suffix | latl- + suffix |
| Can/able | kib- + suffix | layb- + suffix |

```
Kit aloho           = There is God
Kitl-i kṯowo       = I have a book (lit: "there-is-to-me a book")
Latl-i zabno        = I don't have time
Kib-i d-oṯe-no     = I can come
```

## Adjective Agreement

Adjectives follow the noun and agree in gender:

| English | Masculine | Feminine |
|---------|-----------|----------|
| big | rabo | rabṯo |
| small | nacimo | nacimto |
| good | ṭowo | ṭowto |
| bad | ḥarbo | ḥarbto |
| new | ḥaṯo | ḥaṯto |
| old | catiqo | catiqto |
| fast | xayifo | xayifto |
| beautiful | šafiro | šafirto |

```
u-bayto rabo     = the big house (m.)
i-aṯto šafirto   = the beautiful woman (f.)
a-bote rabe       = the big houses (pl.)
```
